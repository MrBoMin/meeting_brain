# MeetingBrain

AI-powered mobile meeting intelligence app built with React Native (Expo), Supabase, and Google Gemini.

## Features

- Audio recording with multi-language support (Burmese + English)
- AI transcription via Google Gemini
- AI analysis: summaries, action items, decisions, open questions
- Knowledge graph with vector embeddings
- Semantic search across meetings
- Calendar view with meeting scheduling
- Multi-tenant organization system with role-based access
- Modern minimalist UI with 3D depth design

## Tech Stack

- **Mobile**: React Native, Expo Router, React Query
- **Backend**: Supabase (Auth, PostgreSQL, Storage, Edge Functions)
- **AI**: Google Gemini (transcription, analysis, embeddings)
- **Database**: PostgreSQL with pgvector for semantic search
